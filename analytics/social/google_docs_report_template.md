## 2. Instagram 성과

| 항목 | 값 | 해석 |
|---|---:|---|
| Reach |  | 도달 규모 |
| Likes |  | 가벼운 반응 |
| Comments |  | 대화 유도 |
| Saves |  | 보존 가치 |
| Shares |  | 확산 가능성 |
| Followers Gained |  | 계정 성장 기여 |

### 해석 가이드

- 저장률이 높으면 다시 볼 가치가 있는 콘텐츠로 판단한다.
- 공유율이 높으면 다른 사람에게 전달할 만한 정보성이 있었다고 판단한다.
- 댓글 수와 댓글 품질은 다음 Threads 질문 설계에 반영한다.

## 3. Threads 성과

| 항목 | 값 | 해석 |
|---|---:|---|
| Views |  | 노출 규모 |
| Likes |  | 기본 반응 |
| Replies |  | 대화 유도 |
| Reposts |  | 재확산 |
| Quotes |  | 맥락 확산 |
| Shares |  | 외부 전달 |

### 해석 가이드

- Threads는 종합 요약 1개가 아니라 뉴스 카드 1개당 게시물 1개로 평가한다.
- Reply Rate가 높은 게시물은 다음 콘텐츠 질문 후보로 기록한다.
- Reposts와 Quotes는 뉴스의 확산성을 판단하는 핵심 지표로 본다.
